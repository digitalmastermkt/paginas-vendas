A arquitetura da MAIA

No centro, a orquestradora (Claude Code). Ao redor, o bot do Telegram, as skills da base, a memória em arquivo e as integrações de IA. As bordas tracejadas são upgrades que plugam depois, sem tocar no núcleo.

Não é chatbot de FAQ. É orquestradora.

Ela organiza, executa tarefas e devolve a entrega pronta direto no seu Telegram, em qualquer formato.

Como instalar

Da VPS ao "oi" em 5 passos

O instalador é idempotente: pode rodar de novo sem quebrar nada. Passo a passo completo no INSTALL-CLAUDE.md.

Passo 01

Clone na sua VPS

VPS Ubuntu/Debian limpa, com acesso root.

bash
# clonar a base
git clone https://github.com/digitalmastermkt/maia-aluno.git
cd maia-aluno
Passo 02

Rode o instalador

Cuida de timezone, pacotes (Node 22, Claude Code CLI, Python 3.12, ffmpeg, tmux), swap, usuário maia, copia tudo pra /opt/MAIA, cria o venv e instala a unit systemd desabilitada.

bash
sudo bash install.sh
Passo 03

Preencha as credenciais

Obrigatórios: TELEGRAM_BOT_TOKEN, ALLOWED_USERS, ADMIN_CHAT_ID, GEMINI_API_KEY. Lista completa comentada no bot/.env.TEMPLATE.

bash
nano /opt/MAIA/bot/.env
Passo 04

Login na conta Claude + suba o bot

Faça o /login OAuth na sessão tmux e habilite o serviço.

bash
sudo -u maia tmux new -s maia-master
cd /opt/MAIA && claude --dangerously-skip-permissions
# /login pela URL, depois Ctrl-b d
systemctl enable --now maia-telegram-bot
Passo 05

Mande "oi" no seu bot do Telegram

A MAIA conduz o onboarding de marca: pergunta sobre você, o negócio, produtos, tom de voz e canais, e grava tudo em brand/brand.json. A partir daí, ela é a sua IA.
